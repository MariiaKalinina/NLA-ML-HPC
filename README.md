# Numerical Computing & Machine Learning & High Performance Computing

A collection of projects and implementations focusing on Numerical Linear Algebra, Machine Learning, and High Performance Computing.


### 1. Numerical Linear Algebra
- Matrix decompositions (LU, QR, SVD, Eigen)
- Linear systems solving
- Iterative methods
- Sparse matrix computations
- Numerical stability and conditioning

### 2. Machine Learning
- Supervised and unsupervised learning algorithms
- Neural networks and deep learning
- Model evaluation and validation
- Feature engineering and selection
- Optimization techniques

### 3. High Performance Computing
- Parallel computing (OpenMP, MPI)
- GPU acceleration (CUDA, OpenCL)
- Distributed systems
- Performance optimization
- Memory hierarchy optimization
